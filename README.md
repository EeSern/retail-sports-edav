# retail-sports-edav
A data-driven solution developed to identify and rectify operational inefficiencies in a national sports retail chain.

## Key Features:

Data Engineering: T-SQL scripts for complex data staging, handling 150k+ sales records and 50k+ customers.

Advanced Analytics: Implementation of Pearson Correlation to identify pricing power and dynamic correlation between unit price and volume.

Strategic Insights: Actionable recommendations that addressed a 60% late order rate and a -6% margin loss in specific product categories (Golf).

Full-Stack BI: From SQL data cleansing to a multi-page Power BI dashboard featuring executive summaries and "Analysis Sandboxes."
